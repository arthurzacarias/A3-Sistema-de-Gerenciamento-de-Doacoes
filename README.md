Repositório para 

Para executar

1. Pegar o caminho da pasta que o app está
2. Abrir o CMD
3. Usar o comando: cd 'caminho da pasta' e ENTER
4. Baixar as dependencias usando: pip install -r requirements.txt
5. Rodar o servidor, usando: python -m uvicorn app.main:app --reload
6. Abrir o navegador com o link: http://127.0.0.1:8000
